# Dr.Claw 文档索引

## 命名约定

`docs/` 下运维文档统一为：

```text
DRCLAW_<主题>[_子主题]_zh.md
```

| 规则 | 说明 |
|------|------|
| 前缀 | 固定 `DRCLAW_` |
| 主题 | 大写蛇形英文（如 `ENV`、`OPENIM_CHANNEL`、`CUSTOMIZATION`） |
| 语言 | 中文文档后缀 `_zh` |
| 标题 | 文内一级标题统一为 `Dr.Claw <中文主题>` |
| 索引 | 链接文案用中文主题名，不用文件名 |
